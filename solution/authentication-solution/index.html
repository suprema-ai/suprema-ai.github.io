QFace Core

QFace Core is Suprema AI’s artificial intelligence face recognition algorithm solution.

What is a face recognition algorithm?

By combining the face detection part that finds a face in the image input through a given device, the facial feature detection that extracts unique features from the detected face, and the face matching part that compares these features with the previously registered features to determine whether they are the same person. It’s called a face recognition algorithm. Suprema AI’s face recognition solution additionally includes face tracking to help quickly detect faces, face grouping to help face matching, forgery detection algorithms to enhance security, and facial features that predict age, gender, race, and expression from a face We provide solutions that help develop more diverse services by adding predictive algorithms.

Advantages of QFace Core

AI algorithm based on deep learning

  • Retains large-capacity face data in Korea and abroad (25 countries, 10,000 people, 50M Images)
    Suprema AI has 50M videos including open face data as well as self-configured face data. These videos include various poses, age, external light, and hidden images for one person, so that they are trained to be strong in various environments when developing a deep learning algorithm.

  • Accurate data labeling with its own superior cleansing technology
    Suprema AI made an excellent contribution to reducing the false detection rate when developing deep learning algorithms by accurately labeling data based on its own database cleansing technology.

  • Deep learning-based face recognition solution using collected data
    Suprema AI’s deep learning-based face recognition algorithm is created using a large amount of cleansed data. Not only can it respond to various situations accurately and accurately, but it is also possible to create algorithms that are robust to a specific environment through various combinations of well-classified DBs as needed.

Fast and accurate recognition performance

  • Excellent detection/matching performance Suprema AI’s face recognition algorithm is fast and accurate. Due to the lightweight deep learning network, not only face detection of 0.01 seconds, but also 1:1 matching and 1:N matching, high-speed matching of 1/600,000 seconds is possible, and it is an excellent algorithm that boasts 99.9% performance at FAR 1/10,000,000.

  • Algorithm robust against external environment variables such as pose/external light
    Suprema AI’s face recognition algorithm was trained using various images (poses, external light, glasses) of one person as well as many people during network training. Due to this, Suprema AI’s face recognition algorithm secured even matching performance in various poses and various lighting environments.​

  • Provide single/multi-face recognition algorithm using Visual / IR image
    Suprema AI’s face recognition algorithm supports not only visual images but also IR images. When it is impossible to recognize only visual images in the night environment, Suprema AI’s algorithm boasts excellent performance even when matching using IR, and it can be used in various fields through multi-face recognition algorithms.

High Security

  • Enhance security through deep learning-based forgery detection algorithm
    In addition to security based on matching performance, Suprema AI boasts high security performance by blocking various forgery faces such as printed paper faces, faces displayed on displays, and 2D masks through the development of a forgery detection algorithm.

  • Enhanced forgery detection through facial component movement
    In addition to judging by the image itself, Suprema AI We have a more powerful forgery detection algorithm by detecting the movement of facial components such as blinking, mouth change, and head direction change, or by checking whether a specific action is performed after commanding it.
