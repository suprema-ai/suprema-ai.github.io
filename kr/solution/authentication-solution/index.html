바이오인식 AI 솔루션

클라우드 서버 및 임베디드 시스템상에서 동작 할 수 있는 세계 최고 수준의 AI 얼굴인식 알고리즘 및 솔루션을 제공합니다

슈프리마에이아이(이하, 슈프리마AI)의 바이오인식 AI 솔루션에 대하여 알아보세요.

QFace Core QFace Edge Module

QFace Core

QFace Core는 슈프리마AI의 인공지능 얼굴인식 알고리즘 솔루션입니다.

얼굴인식 알고리즘 이란?

주어진 기기를 통해 입력된 영상에서 얼굴을 찾는 얼굴 검출 부분과 검출된 얼굴에서 고유의 특성을 뽑아내는 얼굴 특성검출, 이 특성들을 기존에 등록된 특성들과 비교하여 동일인인지 판단하는 얼굴 매칭 부분을 합하여 얼굴인식 알고리즘이라고합니다. 슈프리마AI의 얼굴 인식 솔루션은 추가로 얼굴 검출을 빠르게 도와주는 얼굴 추적, 얼굴 매칭을 도와주는 얼굴 그룹화, 보안성 강화를 위한 위변조 감지 알고리즘, 얼굴에서 나이, 성별, 인종, 표정등을 예측하는 얼굴 특징 예측 알고리즘을 추가하여 보다 다양한 서비스 개발에 도움이 되는 솔루션을 제공합니다.  

QFace Core의 특장점

딥러닝 기반의 AI 알고리즘

  • 국내/국외의 대용량 얼굴 데이터 보유 (25개국, 1만명, 50M Images)
    슈프리마AI는 Open face data 뿐만 아니라 자체적으로 구성한 face data 를 포함하여 50M 의 영상을 보유하고 있습니다. 해당 영상들은 한사람에 대해 다양한 포즈, 연령, 외부광, 가려진 영상이 포함되어 딥러닝 알고리즘 개발시 다양한 환경에서도 강인하도록 훈련되어집니다. 

  • 자체 우수 클린징 기술을 보유하여 정확한 데이터 라벨링
    슈프리마AI는 자체 Database cleansing 기술을 기반으로 data들을 정확하게 라벨링하여 딥러닝 알고리즘 개발시 오검출률 감소에 우수한 기여를 하였습니다.

  • 수집된 데이터를 이용한 딥러닝 기반 얼굴인식 솔루션
    슈프리마AI의 딥러닝 기반얼굴 인식 알고리즘은 대용량의 클린징된 데이터를 이용하여 만들어집니다. 정확하고 다양한 상황에 대응될 뿐만 아니라 잘 분류된 DB들을 필요에 따라 다양한 조합을 통해 특정 환경에 강인한 알고리즘들을 만들어 낼 수도 있습니다

빠르고 정확한 인식 성능

  • 우수한 검출/매칭 성능 보유
    슈프리마AI의 얼굴 인식 알고리즘은 빠르고 정확합니다. 경량화된 딥러닝 네트워크로 인하여 0.01초의 빠른 얼굴 검출 뿐만 아니라 1:1매칭, 1:N매칭에서도 1/600,000초의 고속 매칭이 가능하며, FAR 1/10,000,000에서 99.9%의 성능을 자랑하는 우수한 알고리즘 입니다.

  • **포즈/ 외부광 등의 외부 환경 변수에 강인한 알고리즘 **
    슈프리마AI의 얼굴 인식 알고리즘은 네트워크 훈련시 많은 사람뿐만 아니라 한사람의 다양한 영상(포즈, 외부광, 안경)을 이용하여 훈련하였습니다. 이로 인해 슈프리마AI의 얼굴 인식 알고리즘은 다양한 포즈, 다양한 조명환경에서도 고른 매칭 성능을 확보하였습니다.​

  • Visual / IR 영상을 활용한 단일/다중 얼굴 인식 알고리즘 제공
    슈프리마AI의 얼굴 인식 알고리즘은 Visual 영상 뿐만 아니라 IR 영상도 지원합니다. 야간 환경에 Visual 영상만으로 인식이 불가능 할 경우 IR을 이용하여 매칭하는 경우에도 슈프리마AI의 알고리즘은 우수한 성능을 자랑하며, 다중 얼굴 인식 알고리즘을 통해 다양한 분야에도 활용이 가능합니다.

높은 보안성

  • 딥러닝 기반의 위변조 검출 알고리즘을 통한 보안성 강화
    슈프리마AI는 매칭 성능 기반의 보안성 이외에도 위변조 검출 알고리즘 개발을 통하여 프린트된 종이 얼굴, 디스플레이에 나타난 얼굴, 2D 마스크 등 다양한 위변조 얼굴을 막아 높은 보안 성능을 자랑합니다.  

  • 얼굴 구성 요소 움직임을 통한 위변조 검출 강화
    슈프리마AI는 영상 자체로 판단하는 것 이외에 눈의 깜빡임, 입의 변화, 머리 방향 변화 등 얼굴 구성 요소들의  움직임을 검출하거나, 특정 동작을 명령 후 동작을 수행하는지 보는 방법을 통하여 보다 더 강력한 위변조 검출 알고리즘을 보유하고 있습니다. 

QFace Edge Module

QFace Edge Module은 슈프리마AI의 인공지능 얼굴인식 알고리즘인 QFace Core가 탑재된 임베디드 모듈입니다. QFace Edge Module을 사용하면 손쉽게 귀하의 제품에 얼굴인식 기능을 구현할 수 있습니다.
