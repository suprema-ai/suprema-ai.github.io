QFace Engine

QFace Engine은 슈프리마AI의 인공지능 얼굴인식 알고리즘 솔루션입니다.

Q-Face Engine 의 특장점

  • 딥러닝 기반의 AI 알고리즘

    국내/국외의 대용량 얼굴 데이터 보유 (25개국, 1만명, 50M Images)

    자체 우수 클린징 기술을 보유하여 정확한 데이터 라벨링

    수집된 데이터를 이용한 딥러닝 기반 얼굴인식 솔루션

  • 빠르고 정확한 인식 성능

    우수한 검출/매칭 성능 보유

    포즈/ 외부광 등의 외부 환경 변수에 강인한 알고리즘

    Visual / IR 영상을 활용한 단일/다중 얼굴 인식 알고리즘 제공

  • 높은 보안성

    딥러닝 기반의 위변조 검출 알고리즘을 통한 보안성 강화
    얼굴 구성 요소 움직임을 통한 위변조 검출 강화

Q-Face Module

Q-Face Module은 슈프리마AI의 인공지능 얼굴인식 알고리즘인 Q-Face Engine이 탑재된 임베디드 모듈입니다. Q-Face Module을 사용하면 손쉽게 귀하의 제품에 얼굴인식 기능을 구현할 수 있습니다.

Q-Face Module 특장점

  • 세계최고수준의 임베디드 얼굴인식 성능 제공

  • 손쉬운 시스템 통합 개발

  • 높은 보안성
